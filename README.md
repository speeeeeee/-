# writing-stop-Prevention
제작자: 이네르스 대신 업로드함 

설치법
1. zip으로 다운로드한다
2. zip파일 압축을 푼다
3. chrome://extensions에 들어간다
4. 개발자 모드를 켠다
5. 압축 풀린 파일 로드를 클릭후 파일을 선택한다
6. 즐긴다
